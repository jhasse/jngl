Just a Neat Game Library

Easy to use C++ game library for Linux, Windows, macOS, Android, iOS, Xbox, the Nintendo Switch1, LG webOS and the Web.

  • GitHub
  • Try JNGL in your browser

Features

  • C++17
  • zlib License, so suitable for commercial use and static linking
  • OpenGL (ES) 3.0 accelerated
  • PNG, BMP and WebP image support
  • Ogg Vorbis audio support with support for effects like pitch shifting
  • Theora video playback support
  • UTF-8 text output using FreeType
